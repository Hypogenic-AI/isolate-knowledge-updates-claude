# Downloaded Papers

This directory contains academic papers relevant to knowledge editing and isolated model updates research.

## Core Papers (Essential Reading)

### 1. ROME: Locating and Editing Factual Associations in GPT
- **File**: `2202.05262_Locating_and_Editing_Factual_Associations_in_GPT.pdf`
- **Authors**: Kevin Meng, David Bau, Alex Andonian, Yonatan Belinkov
- **Year**: 2022 (NeurIPS)
- **arXiv**: 2202.05262
- **Why relevant**: Foundational paper introducing ROME method for localized fact editing. Introduces causal tracing and rank-one model editing. Essential for understanding knowledge localization.

### 2. Memory-Based Model Editing at Scale (MEMIT predecessor)
- **File**: `2206.06520_MemoryBased_Model_Editing_at_Scale.pdf`
- **Authors**: Eric Mitchell, Charles Lin, et al.
- **Year**: 2022
- **arXiv**: 2206.06520
- **Why relevant**: Introduces SERAC approach for memory-based editing.

### 3. Model Editing at Scale leads to Gradual and Catastrophic Forgetting
- **File**: `2401.07453_Model_Editing_at_Scale_leads_to_Gradual_and_Catast.pdf`
- **Authors**: Akshat Gupta, Anurag Rao, Gopala Anumanchipalli
- **Year**: 2024
- **arXiv**: 2401.07453
- **Why relevant**: CRITICAL - Shows that edits are NOT as local as believed. Documents gradual and catastrophic forgetting. Directly challenges feasibility of perfect isolation.

### 4. EasyEdit Framework
- **File**: `2308.07269_EasyEdit_An_Easytouse_Knowledge_Editing_Framewo.pdf`
- **Authors**: Peng Wang, Ningyu Zhang, et al.
- **Year**: 2023
- **arXiv**: 2308.07269
- **Why relevant**: Unified framework for knowledge editing. Will use for experiments.

### 5. Fast Model Editing at Scale (MEND)
- **File**: `2110.11309_Fast_Model_Editing_at_Scale.pdf`
- **Authors**: Eric Mitchell, Charles Lin, et al.
- **Year**: 2021
- **arXiv**: 2110.11309
- **Why relevant**: Meta-learning approach to model editing. Baseline method.
